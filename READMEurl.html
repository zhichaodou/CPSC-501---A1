Build Instructions

  • Install Java.
  • Create a new directory.
  • Copy the src directory to the new directory.
  • Switch to the new created directory you will have:
    • Directory Name
    • |---src
    • |---main
    • |---map
    • |---state
    • |---music
    • |---musicEffects
    • |---images
    • |---bin
  • start by compiling the game:
    • javac -d bin/ -cp src src/main/Main.javac
  • Now it's time to run the game:
    • java -cp bin main/Main
  • Tested on
    • Linux Ubuntu 16.04 LTS
    • MacOs 10.11.3 Mac Users may encounter some problems with the GUI)
    • Windows 10
